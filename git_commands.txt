1.Starting New Project (First Time Setup)
cd path/to/project
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/Ashokreddy67/gitp.git
git push -u origin main

cd path/where/you/want/project
git clone https://github.com/Ashokreddy67/gitp.git
cd gitp
